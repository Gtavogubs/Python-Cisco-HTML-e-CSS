Com o Média Query vamos criar configurações adequadas para cada tela,

Lá em cima no link css (link="stylesheet" href=style.css") vamos inserir o média depois do href (link rel="stylesheet" href="style.css" media="")

Temos alguns tipos de media

media="all" para todos os tipos

media="screen" para telas.

media="print" para impressão

Em screen temos ainda algumas configurações

media="screen and (orientation:landscape)" para páginas na horizontal ou paisagem

media="screen and (orientation:portrait)" para páginas na vertical ou retrato

Podemos também fazer outras configurações mais personalizadas de acordo com esta tabela:

Telas pequenas até 600px

Celular de 600px até 768

Tablet de 768px até 992px

Desktop 992px até 1200px

Utilizamos o media de acordo com a página

@media screen and (min-width:768px)
