Pseudo Classes

Pseudo Classe ela deve estar relacionada a uma classe e ela está diretamente relacionada no (estado) de um elemento ex

Pseudo element está relacionado ao elemento, eles alteram o conteúdo periférico de um elemento

Se ele está marcado

Se ele está vazio

Se ele está desabilitado, entre outros

Para selecionarmos pseudo CLasses utilizamos o (:) mais o estado

Para adicionarmos o pseudo element utilizamos o (>)
